# Tech Context: PreClinic Setup

## Core Stack

### Backend
- **Python 3.10+**
- **FastAPI**: Main web framework.
- **SQLAlchemy**: Database ORM.
- **SQLite**: Local relational database.
- **Uvicorn**: ASGI web server.

### Doctor Web Dashboard
- **React / Next.js**: App Router, SSR, Server Components.
- **Tailwind CSS**: Utility-first CSS styling.
- **shadcn/ui**: Accessible UI primitives (customized).
- **lucide-react**: Icon sets.

### Patient Mobile App
- **React Native (Expo SDK 50+)**
- **Expo Router**: File-based tabs and stacks.
- **React Native Paper**: Material-inspired visual component framework.
- **React Native Vector Icons**: Icon graphics.
